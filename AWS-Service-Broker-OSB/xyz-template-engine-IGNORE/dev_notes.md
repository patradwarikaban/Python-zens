# Setup the Dev Env

##### Linux
```
sudo apt-get install python3-venv    # If needed
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```

##### macOS
```
python3 -m venv env
source env/bin/activate
pip3 install -r requirements.txt
```

##### Windows
```
py -3 -m venv env
env\scripts\activate
pip3 install -r requirements.txt
```
